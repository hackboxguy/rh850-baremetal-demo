# Device Header

This directory contains the Renesas device header for the R7F701686 (RH850/F1KM-S1).

## dr7f701686.dvf.h

This file is provided by Renesas Electronics Corporation and is subject to their
license terms. It is intended for use with Renesas products only.

If you do not have this file, you can generate it using:
1. Renesas e2 studio with Smart Configurator
2. Select target device R7F701686
3. Export the device header (`dr7f701686.dvf.h`)
4. Place it in this directory
